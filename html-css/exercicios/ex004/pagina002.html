      Anotomia do Tipo

      metrico

      Letra x = Define o todas as fontes. Define a altura base de todas as letras. A distância das duas linhas do x - Se chama altura x.

      A linha que de define as letras maiúculas - Se chama altura maiúsculas. A linha da altura das letras maiúscula - Se chama ascendente.

      A linha que de define as letras vazadas para baixo - Se chama decedente.

      todas as letras do da fonte - se chama corpo.

      zeometrioco

      serifa é um pequeno tracinho que gera no final da fonte.

      k = tem duas haste / A = tem um haste deitada que uni duas hastes - se chama filete.

      um filete uni duas haste ou um ou mais hastes ou arco

      n = tem duas haste e um barriga qu é arco

      b = tem saltinho - segurar a letra - é chamdo de esporão.

      A = tem a parte de cima - se chama vétice.

      r = a ponta de cima - se chama terminal.

      corpo

      k = base que sai de um uma haste e se leva para cima - se chama braço. base que sai de uma haste ou braço que vai para baixo - se chama perna. de babixo da perna tem o pé que as vezes pode ter serifa.

      s = curva - se chama espinha

      b = haste sai dois arco junto se chama barriga.

      o burraco na letra - se chama olho

      g = o ponto da parte de cima se chama orelha. arte de baixo - se chama calda

      glifo é uma letra

      todas as letras do alfaberto são glifo. conjuto de todos os glifos são fontes.
